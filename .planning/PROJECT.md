# Project: NSEBOT
NSE Option Chain Anomaly Detection Bot with Chrome Extension and HTTP Bridge.

## Context
A system designed to monitor NSE option chains, detect spikes in OI and Price, and alert the user via Telegram and a Chrome Extension popup.

## Stack
- Python (Flask, BeautifulSoup/NSE API)
- SQLite (Persistence)
- JavaScript (Chrome Extension MV3)
- Streamlit (Dashboard)
